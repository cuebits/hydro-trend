About

HydroTrend is a web application for uploading, browsing, downloading, and analysing rainfall data around the globe. It was built as a final project for the Harvard CS50X Computer Science course, and to facilitate independent climate research for UNSW. It is open source and a work in progress, so please feel free to contribute to the codebase at this repository: https://github.com/cuebits/hydro-trend.
